## Descripción

`huevoScan_77` es un script de Bash diseñado para realizar tareas de **reconocimiento y análisis web** de manera automática directamente desde Termux. Ideal para profesionales y entusiastas de la ciberseguridad que necesitan un flujo rápido y completo de información sobre dominios y servicios.

---

## Características

- Escaneo de **subdominios** usando `subfinder` y `amass`.
- Escaneo completo de **puertos y servicios** con `nmap`.
- Identificación de **tecnologías web** usando `whatweb`.
- Captura de **screenshots** de sitios web con `gowitness`.
- Reconocimiento **automático completo**, ejecutando todas las tareas anteriores en un solo comando.
- Traductor de texto al **español** usando `trans`.
